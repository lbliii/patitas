# 0.3.2

**Released**: 2026-03-01

LLM-safe rendering, composable sanitization policies, text extraction, and parallel benchmark.

---

## Highlights

- **LLM renderer** — `render_llm()` outputs structured plain text for model consumption. No HTML; explicit labels for code, math, images. Skips raw HTML for safety.
- **Sanitization** — `sanitize(doc, policy)` with composable policies. Strip HTML, dangerous URLs, zero-width/bidi chars (Trojan Source mitigation). Pre-built: `llm_safe`, `web_safe`, `strict`.
- **`extract_text()`** — Extract plain text from any AST node. Skips HtmlBlock/HtmlInline. Used for slugs, excerpts, LLM pipelines.
- **Parallel benchmark** — `benchmarks/benchmark_parallel.py` demonstrates near-linear free-threading scaling (e.g. 6.6x on 8 cores).

---

## Features

### LLM Renderer

```python
from patitas import parse, render_llm

doc = parse("# Hello **World**\n\n- item\n\n```python\nx = 1\n```")
render_llm(doc)
# '# Hello World\n\n- item\n\n[code:python]\nx = 1\n[/code]\n\n'
```

### Sanitization

```python
from patitas import parse, sanitize
from patitas.sanitize import llm_safe, web_safe

doc = parse("# Title\n\n<script>alert(1)</script>\n\n[link](javascript:void(0))")
clean = sanitize(doc, policy=llm_safe)  # Strips HTML, dangerous URLs, zero-width chars
```

Compose policies with `|`:

```python
from patitas.sanitize import strip_html, strip_dangerous_urls, normalize_unicode

custom = strip_html | strip_dangerous_urls | normalize_unicode
```

### Extract Text

```python
from patitas import parse, extract_text

doc = parse("# Hello **World**")
extract_text(doc.children[0])  # 'Hello World'
```

### LLM Safety Example

```bash
python examples/llm_safety/llm_safe_context.py
```

Shows the full pipeline: parse → sanitize → render_llm for RAG/retrieval contexts.
