# 0.3.4

**Released**: 2026-03-06

Bug fix for directives with `preserves_raw_content=True` (e.g. `list-table`).

---

## Highlights

- **`preserves_raw_content` fix** — Directives like `list-table` that use `preserves_raw_content=True` now receive correct verbatim content. Previously, `_parse_block()` consumed list tokens before `raw_content` could capture them, resulting in empty or partial content.
- **Source-offset extraction** — Uses source offsets to extract verbatim text when `preserves_raw_content` is True, ensuring list-like directive content is preserved correctly.
